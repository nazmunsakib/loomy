Loomy is a high-performance, modern WooCommerce theme built with Tailwind CSS 4 and Alpine.js.

== Description ==

Loomy is designed for speed and flexibility. It integrates deeply with WooCommerce and Elementor while maintaining a lean codebase.

== Installation ==

1. In your WordPress dashboard, go to Appearance > Themes > Add New.
2. Upload the loomy.zip file.
3. Activate the theme.
